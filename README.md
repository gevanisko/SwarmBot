# SwarmBot

ESP32-S3-based swarm robot PCB designed for small autonomous robotics, camera input, wireless communication, and servo motor control.

This was the first PCB I designed, making it an important learning project in schematic design, PCB layout, power regulation, high-speed memory routing, camera interfacing, and embedded hardware integration.

## Overview

SwarmBot is a custom ESP32-S3 robot controller board intended for use in a small swarm robotics platform. The board combines an ESP32-S3 microcontroller with external memory, camera support, servo outputs, Bluetooth communication, USB-C, and onboard power regulation.

The design focuses on creating a compact embedded controller that can support camera-based sensing, wireless communication between robots, and motor/servo control for robotic movement.

## Features

- ESP32-S3-based robot controller
- Quad SPI external flash
- External SRAM / PSRAM
- DVP camera interface through an FFC connector
- 4 servo motor output pins
- Bluetooth antenna for wireless communication
- USB-C connection for programming, power, and debugging
- Dedicated power regulation circuitry
- Modular Altium schematic structure
- First custom PCB design project
